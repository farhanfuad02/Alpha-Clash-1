# Alpha Clash Pro

**Alpha Clash Pro** is an interactive web-based typing game designed to help users improve their keyboard typing speed and accuracy. With a modern, dark-themed UI and visual feedback, it turns typing practice into an engaging "Keyboard Kung Fu" session.

## ‚ú® Features

- **Interactive Gameplay**: Type the random alphabet displayed on the screen to score points.
- **Visual Feedback**: The virtual keyboard highlights the target key.
- **Score & Lifeline System**: Track your progress with a dynamic scoring system and limited lifelines.
- **Responsive Design**: Built with a "Mobile-First" approach using standard artboard sizes, but playable on larger screens.
- **Modern UI/UX**: distinct "Glassmorphism" effects, gradients, and custom fonts.

## üöÄ How to Play

1. **Start the Game**: Click the "Play Now" button on the home screen.
2. **Type the Alphabet**: A random letter will appear on the screen. Press the corresponding key on your physical keyboard.
3. **Score Points**: fast and correct typing increases your score.
4. **Game Over**: If you miss keys or run out of time/lives (implementation dependent), the game ends and shows your final score.
